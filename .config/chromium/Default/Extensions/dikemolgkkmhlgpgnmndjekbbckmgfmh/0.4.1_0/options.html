Volume Controller
Enable extension in youtube.com
Show audio tabs list by default
Volume step size (via keyboard controls)
What's new?
  • v0.4 - Configure volume step size
  • v0.3 - Keyboard shortcuts
    • Create keyboard shortcut to open extensions popup
      chrome://extensions/ -> scroll to bottom -> select volume controller extension
      then press the shortcut key you want to keep, preferrebly alt + dot or alt + →
      Do not use global shortcuts like ctrl + c or ctrl + p
      Now you can control volume without need for mouse/touchpad
    • alt + → then press m to toggle mute, ↑ to increase and ↓ to decrease volume
Note:
  • Flash videos volume cannot be controlled using this extension, only HTML5 videos and audios.
  • Few sites load the video from different origin, for example voot.com. Those videos cannot be accessed/controlled because of Same-origin policy.
  • You can always override the volume of any video by the controls given from the website. But that volume is not reflected in the extension, as extenstion saves the global value.
  • Incognito mode: Websites will not load user saved settings in this mode, even the volume which you last set. So allow this extension to run in incognito mode to manage volume.
    • Steps: Right click on extenstion
      >>
      manage extensions
      >>
      tick mark the checkbox which says "Allow in incognito".
